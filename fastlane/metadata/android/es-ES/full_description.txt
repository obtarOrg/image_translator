#OCR

Una aplicación basada en Tesseract para el reconocimiento de texto en imágenes.

Esta aplicación actualmente (desde v3.0) se basa en <a href="https://github.com/tesseract-ocr/tesseract/blob/master/README.md">Tesseract 5</a> y es la primera basado en Teseracto 5.

###Después de descargar los datos de entrenamiento, la aplicación hace todo sin conexión.

Esta aplicación se puede traducir en [Hosted Weblate](https://hosted.weblate.org/projects/android-ocr).

🚀 <font color="#2467d6"><b>Leistungsmerkmasculino</b></font>
* Máxima precisión y resultados más rápidos
* Extraer textos de imágenes
* Copiar datos al portapapeles
* Seleccione cualquier parte de los datos
* Procesar varios idiomas en una imagen (se necesitan cambios de configuración)
* Basado en los últimos <a href="https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files.md">datos de capacitación</a>
* Más de 120 <a href="https://tesseract-ocr.github.io/tessdoc/Data-Files">idiomas</a> reconocidos
* 3 tipos de datos, dh <a href="https://github.com/tesseract-ocr/tessdata">Estándar</a>, <a href="https://github.com/tesseract-ocr/tessdata_best" >Bestens</a>, <a href="https://github.com/tesseract-ocr/tessdata_fast">Schnell</a>, weitere Informationen dazu gibt es <a href="https://github.com /tesseract-ocr/tessdoc/blob/master/Data-Files.md">aquí</a>
* Reconocimiento de caracteres matemáticos y ecuaciones (aún no probado)
* Procesamiento de imágenes directamente desde la galería a través del menú para compartir


 <font color="#2467d6"><b>Permisos necesarios</b></font>

* Internet: Para descargar los datos de entrenamiento
* Acceso a Internet: para descargar los datos de entrenamiento
*Nota: Ninguno de sus datos se envía a ninguna parte
